# Gestión de solicitudes

## Descripción

Hice este programa en C# para una práctica. Sirve para registrar solicitudes de clientes y manejar sus estados sin errores usando enum.

## Enum

Usé un enum llamado EstadoSolicitud con:
Pendiente, EnProceso, Completada y Cancelada.
Así no se pueden poner estados mal escritos.

## Funciones

Se puede:

* Registrar solicitudes
* Verlas
* Cambiar el estado
* Buscar por ID

## Nota

Las solicitudes empiezan en Pendiente y el estado se cambia con opciones del menú.

## Ejecución

Se corre en consola desde Visual Studio.
